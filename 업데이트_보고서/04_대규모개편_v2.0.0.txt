효과:

  ✓ 향후 추가 페이지 확장 용이


2.2 페이지 구성 및 기능
────────────────────────

[페이지 1] 📄 문서 챗봇
  
  위치: 첫 번째 탭
  
  주요 기능:
    • PDF 문서 업로드 (중앙 카드 배치)
    • 문서 기반 RAG 챗봇 Q&A
    • 회전하는 지구 이모지 애니메이션
    • 시스템 설정 (출처 강조, 신뢰도 표시, 쿼리 확장)
    • 고급 옵션 (expander)
    • 대화 기록 초기화 버튼
  
  UI 개선:
    • 파일 업로드 카드를 메인 제목 아래에 배치
    • 설정이 메인화면 상단에 표시되어 한눈에 파악 가능
    • 시스템 설정 토글이 2열 배치로 공간 효율화
    • 기존 대비 더 깔끔하고 직관적인 레이아웃


[페이지 2] 📝 보고서 작성기
  
  위치: 두 번째 탭
  
  주요 기능 (프로토타입):
    • 보고서 제목 입력
    • 보고서 유형 선택 (프로젝트/성과/분석/재정/월간/기타)
    • 주요 내용 요약 입력
    • 핵심 성과 및 지표 입력
    • "AI로 보고서 작성" 버튼
    • 생성된 보고서 마크다운 다운로드
    • 편집 모드, 다시 생성 옵션
  
  사용 시나리오:
    - 임직원들이 정형화된 형식으로 빠르게 보고서 작성 가능
    - AI가 자동으로 구조화된 보고서 생성
    - 개인 맞춤형 보고서 다운로드 지원


[페이지 3] ✉️ 이메일 Assistant
  
  위치: 세 번째 탭
  
  주요 기능 (프로토타입):
    • 이메일 목적 선택 (정보 공유/회의 요청/프로젝트 제안/상태 보고/협업 요청/기타)
    • 수신자 입력
    • 이메일 내용 요약 입력
    • 톤&스타일 선택 (격식적/친근한/긴급/정중한/캐주얼)
    • "AI로 이메일 작성" 버튼
    • 생성된 이메일 다운로드
    • 편집, 다시 작성 옵션
  
  사용 시나리오:
    - 다양한 목적의 이메일을 빠르게 작성 가능
    - 톤에 맞는 인사말과 종료 인사 자동 생성
    - 상황별 최적화된 이메일 템플릿 제공


2.3 UI/UX 개선사항
────────────────────────

전체 레이아웃:
  ✓ 상단 헤더 재설계 (🌎 월드비전 AI Suite 로고)
  ✓ 3개 페이지 네비게이션 버튼 (중앙 정렬)
  ✓ 가로줄(---) 구분선으로 시각적 계층화
  ✓ 메인 컨텐츠 영역 최적화

색상 체계:
  • 헤더: 어두운 그래디언트 배경 (남색 → 검정)
  • 버튼: 흰색 배경 + 주황색 테두리 (호버 시 주황색 배경)
  • 텍스트: 흰색 (높은 대비율)
  • 카드: 흰색 배경 + 라이트 그레이 테두리

애니메이션:
  • 회전하는 지구 이모지 (3초 주기)
  • 버튼 호버 시 상향 이동 애니메이션

반응형 디자인:
  • 2열/3열 컬럼 레이아웃으로 모바일/태블릿 대응
  • 모든 입력 폼이 적응형 너비 설정


2.4 기술적 개선사항
────────────────────────

Session State 관리:
  • st.session_state를 통한 전역 상태 관리
  • 페이지 간 이동 시에도 설정값 유지
  • 주요 관리 변수:
    - current_page: 현재 활성 페이지
    - emphasize: 출처 근거 강조 표시 여부
    - show_confidence: 신뢰도 표시 여부
    - enable_query_expansion: 쿼리 확장 활성화 여부
    - current_file_hash: 파일 변경 감지용 해시값
    - messages: 대화 기록 저장

코드 구조:
  • 사이드바 관련 CSS 제거 (현재 불필요)
  • 페이지별 조건부 렌더링 (if-elif 구조)
  • 각 페이지가 독립적으로 동작 가능하도록 설계


2.5 기존 기능 유지
────────────────────────

문서 챗봇 기능 (v1.0.0 이상):
